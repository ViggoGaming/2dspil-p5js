# Usage
Start med at hente koden ned:
```bash
git clone https://github.com/ViggoGaming/2dspil-p5js
```

Gå ind i mappen:
```
cd 2dspil-p5js
```

Installer pakker:
```
npm install --save
```

Start serveren:
```
node app.js
```
